## DUOLINGO CS

This project is a computer science course in the form of Duolingo. Duolingo is a mobile learning software that uses short-form learning techniques to promote easy access to learning. The course is meant for those with little to no computer science experience and uses some Python to cover four fundamental units of computational thinking: algorithms, variables, conditionals, and loops. Each unit has 3-4 lessons which each include 10 questions of simple, interactive, and engaging questions that teach these computational thinking concepts. This project was made to lower the barrier of entry for computer science, showing learners that learning computer science can be done anywhere with only a phone. Many people believe that computer science is a difficult subject only reserved for certain types of people. However, this course allows anyone to dip their toes into the world of computational thinking. The questions take on a theoretical learning perspective where computer science does not always need to be programming, but rather more logical problem-solving applicable to daily life. 




## How to run the project
This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

**but I highly recommend just pressing "W" and viewing it on your webpage!!**
